# DeadMouse 

Ditch your mouse. Surf the web with just your keyboard.

Try it out and get it [here](http://chetan51.github.com/deadmouse/).

## Usage

1. Clone the repo
2. Run `cake watch`
3. Load directory as an extension in Chrome
